# Spend Smart

## Overview
Spend Smart is a personal expense tracking application built using .NET. It is designed to help users manage daily expenses, track spending patterns, and maintain better financial control.

This project is currently under active development and is being expanded as I continue to learn and implement new features.

---

## Features (Planned & In Progress)

- Add, update, and delete expenses  
- Categorize spending (e.g., food, transport, bills)  
- View expense history  
- Basic summaries and insights  
- User-friendly interface  

---

## Tech Stack

- .NET  
- C#  
- SQL Database

---

## Status

This project is a work in progress and will be updated with new features and improvements.

The goal of this project is to strengthen my .NET development skills while building a practical, real-world application.
